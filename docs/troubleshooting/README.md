# Troubleshooting

> Known issues, investigations, and their resolutions.

## Format

Each investigation should be a separate file:

```markdown
# Issue Title

**Status:** open | investigating | resolved | wontfix
**Date:** YYYY-MM-DD
**Related:** link to related issues

## Problem
Clear description of the symptom.

## Hypotheses
1. Possible cause A
2. Possible cause B

## Investigation
What was tried, what the results were.

## Resolution
How it was fixed (if resolved).

## Lessons
What to remember for next time.
```
